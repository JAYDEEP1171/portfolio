LoanLens

Predict loan eligibility with a machine learning-powered web app designed for financial insights.

Project Overview

LoanLens is a Flask-based web application that uses a logistic regression model to predict loan eligibility. It supports both individual and batch predictions with detailed performance metrics.

Technologies: Python, Flask, Scikit-learn, Pandas, NumPy, Tailwind CSS, Render.

Repository: GitHub | Live App: Render

Key Features

Batch Prediction

Upload a CSV file to predict loan eligibility for multiple applicants.

Single Form Input

Enter individual applicant data to get a real-time loan prediction.

Performance Metrics

Displays accuracy, precision, recall, and F1-score for transparency.

Challenges & Solutions

  • Dataset Handling: Loan-train.csv was included in the repo as it fit GitHub's size limit.
  • Consistent Encoding: LabelEncoders were stored to ensure test data matches training format.
  • Startup Delays: Identified as a Render free-tier limitation, noted for future upgrades.

Future Enhancements

  • Implement Random Forest or XGBoost for better accuracy.
  • Enable download of prediction results as CSV.
  • Add login/authentication for secure access.
  • Visualize feature importance with charts.
  • Pre-train and load models with joblib to reduce startup time.
