About the Project

YouSentiment is a Flask-powered web app that analyzes YouTube video comments to reveal audience emotions. Using the YouTube Data API, TextBlob for sentiment analysis, and tools like WordCloud and Matplotlib, it delivers stunning pie charts and word clouds to visualize positive, negative, and neutral sentiments.
